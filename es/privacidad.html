Política de Privacidad

Versión: 1.0  |  Fecha de vigencia: 12 de marzo de 2026  |  Última actualización: 12 de marzo de 2026

Esta es una traducción de cortesia. La versión en inglés es el documento legalmente vinculante.

1. Introducción y Alcance

Esta Política de Privacidad explica cómo Ansuken LLC, una LLC de Florida ("Compañía," "nosotros," "nuestro," o "Ansuken"), recopila, usa, divulga, retiene y procesa datos personales en conexión con la plataforma Vortex CX (la "Plataforma").

Para usuarios de la plataforma: Ansuken es el controlador de datos — usted nos autoriza a procesar su información de cuenta.

Para contactos de clientes (sus clientes): Ansuken actúa como procesador de datos en su nombre. Usted es el controlador de datos responsable de los Datos del Cliente.

2. Información que Recopilamos

  • Información de registro: Nombre, email, teléfono, empresa, cargo
  • Datos de comunicación: Contenido de mensajes (WhatsApp, SMS, email, voz), metadatos, grabaciones de llamadas, transcripciones
  • Datos de contacto: Nombres, teléfonos, emails, direcciones físicas (vía autocompletado de Google Maps) de sus contactos
  • Datos de uso: Funciones accedidas, configuraciones de flujos de trabajo, datos de integración CRM
  • Datos técnicos: Dirección IP, tipo de dispositivo, navegador, cookies
  • Datos de pago: Nombre de facturación, dirección, historial (no almacenamos números de tarjeta completos)
  • Datos de integración: Tokens OAuth, datos sincronizados de CRM, datos de Google Workspace, datos de Telnyx

3. Cómo Usamos su Información

Usamos su información para: proporcionar servicios de la plataforma, procesar y sincronizar datos con sistemas integrados, entregar transcripciones y análisis con IA, generar reportes de comunicación, brindar soporte al cliente, manejar facturación, mantener seguridad y prevenir fraude, y mejorar la plataforma (solo con datos agregados y anonimizados).

4. Cómo Compartimos su Información

NO vendemos sus datos.

Compartimos información con proveedores de servicios que actúan como procesadores:

  • Telnyx — Servicios de SMS y voz
  • Google Workspace (Gmail API) — Envío y recepción de email vía OAuth 2.0
  • Google Maps Platform (Places API) — Autocompletado y validación de direcciones
  • Meta Platforms (WhatsApp) — Integración de mensajería WhatsApp Business
  • Deepgram — Transcripción de voz a texto
  • Google Cloud — Infraestructura de procesamiento y almacenamiento
  • Hostinger — Alojamiento VPS
  • Supabase — Base de datos y autenticación
  • Stripe — Procesamiento de pagos (compatible PCI-DSS)

5. Seguridad de Datos

  • Encriptación en tránsito (TLS 1.2+) y en reposo (AES-256)
  • Bóveda PII — datos personales nunca en tablas operativas
  • Aislamiento de inquilinos mediante Row-Level Security (RLS) a nivel de PostgreSQL
  • Sin PII en logs — solo IDs internos y referencias de bóveda
  • Tokens OAuth encriptados, contraseñas hasheadas con bcrypt
  • Notificación de brechas dentro de 72 horas

6. Sus Derechos de Privacidad

Usted tiene derecho a: acceder, corregir, eliminar, portar sus datos, restringir el procesamiento, retirar consentimiento y objetar al procesamiento. Para todas las solicitudes: privacy@vortex-cx.com. Respondemos dentro de 30 días.

Cumplimos con GDPR (UE), CCPA (California), y LGPD (Brasil).

Divulgación de Servicios API de Google

El uso y transferencia de información recibida de las APIs de Google por parte de Vortex CX cumple con la Política de Datos de Usuario de Servicios API de Google, incluyendo los requisitos de Uso Limitado.

Gmail API: Solo usamos datos de Gmail para los propósitos descritos (servicio de comunicación unificada). No usamos datos de Gmail para publicidad, no permitimos que humanos lean su contenido de email (excepto los agentes autorizados en su cuenta), y no transferimos datos de Gmail a terceros excepto lo necesario para proporcionar el servicio.

Google Maps Platform: Usamos la API de Places para autocompletado de direcciones. Las consultas se envían a Google para obtener sugerencias validadas. Solo la dirección seleccionada se almacena. No se realiza seguimiento de ubicación.
